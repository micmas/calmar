# Retrieval Practice

## RTSS specification

**Target**: Long-term retention of trained word forms; memory consolidation.

**Ingredient**: Repeated test–study–retrieve cycles in which the patient actively attempts recall before feedback is provided, rather than passively restudying the target. In spell-study-spell this is the final 'spell' step after copying.

**Theory**: Testing effect / retrieval practice effect (Roediger & Karpicke 2006): actively retrieving information from memory strengthens the memory trace more than equivalent time spent restudying. In the context of acquired dysgraphia, forced retrieval attempts reconstruct degraded graphemic representations through pattern-completion in surviving orthographic networks.
