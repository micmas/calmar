# Phonological Feature Analysis

## RTSS specification

**Target**: Phonological access; lexical-phonological retrieval for word naming.

**Ingredient**: Clinician-guided systematic analysis of a target word's phonological components from a picture: first phoneme, rhyming word, number of syllables, another word beginning with the same phoneme, and final phoneme. A visual phonological component chart provides orthographic scaffolding. Patient attempts naming after completing each component.

**Theory**: Spreading activation: elaborating the phonological representation of the target word activates multiple phonological nodes simultaneously, increasing the probability that the target lexical-phonological form reaches threshold. Repeated activation of these nodes strengthens lexical-phonological links (MassonTrottier et al. 2021 rsFC data show increased anterior temporal–visual connectivity following Fr-PCA, consistent with strengthened bottom-up lexical-phonological pathways).

## Used in therapies

- phonological_component_analysis


## Addresses impairments

- anomic_aphasia
- anomia
- phonological_production
- form_to_articulation


## Notes

PCA is specifically designed for naming impairment with intact phonological processing; not suitable for phonological impairment at the input level or for severe comprehension deficits. Evidence base is from chronic aphasia with anomia (MassonTrottier 2021, n=10, rs-fMRI).
