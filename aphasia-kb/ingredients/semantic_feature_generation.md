# Semantic Feature Generation

## RTSS specification

**Target**: Semantic access; lexical-semantic retrieval for word naming.

**Ingredient**: Clinician-guided generation of semantic features for a pictured target item across five categories: group (category), use (function), physical properties (colour, shape, size), location, and association. The patient attempts target naming at each step, with feature discussion continuing regardless of naming success.

**Theory**: Spreading activation through the semantic network: activating multiple feature nodes linked to the target concept increases summed activation at the target lexical node, eventually reaching retrieval threshold. Repeated activation of the feature–lexical network strengthens semantic-to-lexical connections over time.
