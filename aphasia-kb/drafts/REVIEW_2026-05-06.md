## Cross-cutting patterns

These show up across many drafts and are worth deciding once rather
than per-draft:

1. **Replication-vs-cohort-overlap conflation.** Several drafts list
   same-lab papers under `replications:` (Fridriksson2018Anatomy →
   Fridriksson2018; Bonilha2014 → Fridriksson2018Anatomy; Yourganov
   2015 ↔ Fridriksson2018). The `provenance.flags` already note
   cohort overlap. Decide whether to keep these in `replications`
   (loader treats as independent evidence) or move to a
   `cohort_overlaps_with:` annotation. The schema doesn't currently
   have such a field; consider a v2.4 addition.

2. **`responder` direction values.** Five findings across four
   therapy drafts use `direction: likely` with `relationship:
   responder`. The schema enum allows `likely_responder` precisely
   for this case. Standardize.

3. **Source-passage truncation.** Several predictor drafts (Hu2025,
   Kristinsson2025, Teghipco2024, Dresang2022) have source_passages
   that begin or end mid-word. These will degrade the
   `annotate_paper.py` color-highlight quality. Replace with full
   sentence-bounded quotes.

4. **`atlases_used: []` mismatch.** Several drafts list `[]` even
   though the paper's Methods explicitly names atlases (Stefaniak →
   HO+AAL; Saur → not stated explicitly but available in SI;
   Marchina → Talairach + healthy-control DTI atlas). Audit.

5. **Page numbering convention.** Some drafts use PDF index, others
   journal pagination. Decide: I recommend journal pagination for
   `paper_section` and PDF index for `source_passages[].page`
   (annotator uses page index to place highlights). Document the
   convention in EXTRACTION_SKILL.md.
